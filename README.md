## Setup and Build Intruction

1. Clone the repository
```bash
   git clone https://github.com/hiruvindajayashanka2001/chrome-extension-learning.git
```
2. Install Dipendencies
```bash
   npm install
```
3. Build Extension
```bash
   npm run build
```
