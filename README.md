##Setup and Build Intruction
1. Clone the repository
```bash
   git clone https://github.com/Y3S2-SE/money-mentor.git
   cd money-mentor/backend
```
2. Install Dipendencies
```bash
   npm install
```
3. Build Extension
```bash
   npm run build
```
